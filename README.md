## About Laravel

Laravel is a web application framework with expressive, elegant syntax. We believe development must be an enjoyable and creative experience to be truly fulfilling. Laravel takes the pain out of development by easing common tasks used in many web projects, such as:

-   [Simple, fast routing engine](https://laravel.com/docs/routing).
-   [Powerful dependency injection container](https://laravel.com/docs/container).
-   Multiple back-ends for [session](https://laravel.com/docs/session) and [cache](https://laravel.com/docs/cache) storage.
-   Expressive, intuitive [database ORM](https://laravel.com/docs/eloquent).
-   Database agnostic [schema migrations](https://laravel.com/docs/migrations).
-   [Robust background job processing](https://laravel.com/docs/queues).
-   [Real-time event broadcasting](https://laravel.com/docs/broadcasting).

Laravel is accessible, powerful, and provides tools required for large, robust applications.

## Instalar dependencias del proyecto

composer install

### Arrancar servidor local

php artisan serve

## Crear nuevos modelos

Para crear nuevos modelos se debe escribir el siguiente comando. Ejemplo con modelo Task:

php artisan make:model Task -mcr

-m migration Crea automáticamente el archivo de migración (create_tasks_table.php).
-c controller Genera un controlador (TaskController.php).
-r resource Hace que el controlador sea tipo Resource, es decir, con los métodos index, store, show, update, destroy.

## Definición de schema de migración

Buscar el fichero generado por el comando de generación del modelo en el directorio database/migrations

### Identificador de tabla

$table->id();

### Foreign key

$table->foreignId('user_id')->constrained()->cascadeOnDelete();

### Campo tipo String

$table->string('title');

### Campo de texto largo con posibilidad de ser NULL

$table->text('notes')->nullable();

### Campo booleano con valor por defecto 'false'

$table->boolean('done')->default(false);
